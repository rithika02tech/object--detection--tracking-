 Object Detection and Tracking System

An AI-based object detection and tracking application developed using Python, YOLOv8, OpenCV, and DeepSORT. The project detects multiple objects in real time and assigns unique IDs to track their movement across video frames.

This project was developed as part of my AI internship to explore computer vision, object detection models, and real-time tracking systems using deep learning techniques.

Overview

The system uses the YOLOv8 model to detect objects from webcam or video input and DeepSORT to track detected objects by assigning unique tracking IDs. The application processes video frames in real time, draws bounding boxes around detected objects, and continuously tracks their movement.

The project demonstrates how AI models and computer vision techniques can be combined to build intelligent surveillance and monitoring systems.
